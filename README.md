# personality-test
누구나 자유롭게 만드는 성향 테스트

- 사용스택: TypeScript, Next.js, Jest, NodeJS - Express FrameWork, Spring-Boot
- 개발환경: Window
- 개발도구: Visual Studio Code
- 프로젝트기간: 22. 02 ~ 22. 03

<br/>

프로젝트 설명
-----------
각양각색의 사람들의 성향을 테스트 할 수 있는 폼을 제공해주는 웹어플리케이션으로, next.js, typescript, react test를 학습해 보고자 만들어본 토이 프로젝트입니다.
